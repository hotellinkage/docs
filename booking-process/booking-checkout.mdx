# Checkout

The `<checkout>` component allows guests to enter personal information and complete payment for their selected room. This is Step 2 of the booking flow.

## Basic Implementation

```html
<div data-mirai-component="checkout"></div>
```

## Complete Page Template

```html
<!DOCTYPE html>
<html>
<head>
    <title>Complete Your Booking</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="robots" content="noindex, nofollow" />
    <link rel="stylesheet" href="https://static.mirai.com/core/index.css" type="text/css" media="all" />
</head>
<body>
    <div data-mirai-id="YOUR_HOTEL_ID"></div>
    <div data-mirai-component="checkout"></div>
    <script type="module" defer src="https://static.mirai.com/core/index.js" 
            data-cookieconsent="necessary"></script>
</body>
</html>
```

<Warning>
Use this component on a dedicated page without additional content. Adding navigation bars or widgets may interrupt the booking experience.
</Warning>
